# 🛒 E-Commerce Website

A **Django + React + MySQL** based E-Commerce Website with shopping cart functionality.

## 🚀 Features
- ✅ **View Products**
- 🛒 **Add to Cart**
- ❌ **Remove from Cart**

## 🛠 Tech Stack
- **Frontend:** React.js
- **Backend:** Django + Django REST Framework
- **Database:** MySQL
- **API Requests:** Axios
